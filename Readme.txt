Before running the program do these steps;

1-Go to c:\ and create a file called "Music Player"

2-In file Music Player create a text file called "musics"

3-Add your musics in Music Player file

//WARNING// File format must be ".wav"

4-Add your music's name in musics text file


--------------------------------------------------------------

How to use;

1 -> First enter your music's name

2 -> Click "Select"

3 -> Then hit "Play"
